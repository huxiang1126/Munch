# Security Policy

## Supported Status

Munch is under active development. Security fixes will be handled on a best-effort basis until a formal release policy is published.

## Reporting a Vulnerability

Please do not open a public issue for sensitive vulnerabilities.

Instead:

1. send a private report to `hx831126@gmail.com`
2. include reproduction details
3. include impact and affected area if known

Please allow reasonable time for triage before public disclosure.

## Scope

Security-sensitive areas include:

- authentication
- admin authorization
- storage access
- provider API keys
- upload pipelines
- prompt execution or injection paths
