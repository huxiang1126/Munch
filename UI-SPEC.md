## 0. 设计原则

| 原则 | 说明 |
|------|------|
| **内容优先** | 模板图片是主角，UI chrome 尽量隐形 |
| **沉浸式暗色** | 纯黑底 (#0f0f10)，所有 UI 元素「浮」在黑色之上 |
| **克制交互** | 信息按需出现——悬停才展示标题，点击才展开面板 |
| **底部驱动** | 核心操作（输入/选模型/出图）集中在底部，手指/鼠标最易触达 |

## 2. [组件 1] 顶部状态栏 Top Bar

### 2.1 布局

```
┌──────────────────────────────────────────────────────────┐
│  🔴 Munch     工作台  历史  │      🔍 搜索     │  ● 256积分  👤 │
└──────────────────────────────────────────────────────────┘
高度: 56px
背景: rgba(15,15,16,0.8) + backdrop-blur(20px)
边框底部: 1px solid rgba(255,255,255,0.06)
z-index: 50
position: fixed, top: 0
```

### 2.2 滚动隐藏/出现交互

```
行为规则：
- 用户向下滚动 > 30px：Top Bar 向上平移隐藏（translateY(-100%)）
- 用户向上滚动（任意距离）：Top Bar 向下平移出现（translateY(0)）
- 页面在最顶部时：Top Bar 始终可见
- 动画: transform 0.3s ease

实现方式:
- 监听 window scroll 事件（throttle 16ms）
- 记录 prevScrollY，比较 currentScrollY
- currentScrollY > prevScrollY → 隐藏
- currentScrollY < prevScrollY → 显示
- currentScrollY <= 0 → 强制显示
```

### 2.3 内部元素

| 位置 | 元素 | 说明 |
|------|------|------|
| 左 | Logo（Munch 文字 + 红色图标） | 点击回首页 |
| 左 | 导航标签：工作台 / 历史 | 当前页高亮，白色文字；非当前页灰色 |
| 中 | 搜索图标（可选，V2） | MVP 可省略 |
| 右 | 积分徽章 `● 256` | 红色圆点 + 积分数字，点击跳转积分页 |
| 右 | 用户头像 | 圆形，32px，点击弹出下拉（设置/退出） |

## 4. [组件 3] 模板瀑布流画廊

### 4.1 Masonry 布局

```
布局方式: CSS Columns（非 Grid，更适合瀑布流不等高）

桌面端 (≥1280px): columns: 4, gap: 6px
平板端 (≥768px):   columns: 3, gap: 4px
手机端 (<768px):    columns: 2, gap: 4px

每个图片卡片:
  - break-inside: avoid（防止列断裂）
  - margin-bottom: 6px（同 gap）
  - border-radius: 4px（微圆角，参考 localbanana 的 rounded-[2px]）
  - overflow: hidden

图片比例:
  - 不强制固定比例
  - 使用图片原始宽高比
  - img 标签: width: 100%, height: auto, object-fit: cover
  - 加载前显示 skeleton（与图片同比例的灰色占位块）
```

### 4.2 图片卡片结构

```html
<!-- 伪代码结构，Codex 请用 React 组件实现 -->
<div class="template-card group relative">
  <!-- 图片 -->
  <img src="..." alt="" class="w-full h-auto rounded-[4px]" />

  <!-- Hover 遮罩 + 标题（默认隐藏） -->
  <div class="
    absolute inset-0
    bg-gradient-to-t from-black/70 via-transparent to-transparent
    opacity-0 group-hover:opacity-100
    transition-opacity duration-200
    flex items-end
    p-3
  ">
    <span class="text-white text-sm font-medium">高端护肤大片</span>
  </div>
</div>
```

### 4.3 Hover 效果详细规范

```
默认状态:
  - 只显示图片，无任何文字/图标
  - 图片 scale(1.0)

Hover 状态（200ms 过渡）:
  - 图片轻微放大: scale(1.02)
  - 底部出现渐变遮罩（从底部 30% 开始，黑色 70% 不透明度）
  - 遮罩内底部左对齐显示模板名称（白色，14px，font-medium）
  - cursor: pointer

点击反馈:
  - scale(0.98) 持续 100ms，然后弹回
  - 触发模板详情弹窗（组件 5）
```

### 4.4 分类筛选（顶部粘性标签栏）

```
画廊区域顶部，粘性定位:

┌──────────────────────────────────────────────────┐
│  全部  护肤美妆  服装穿搭  食品饮品  通用产品  海报广告  │
└──────────────────────────────────────────────────┘

位置: sticky, top: 0（Top Bar 隐藏时）或 top: 56px（Top Bar 显示时）
背景: rgba(15,15,16,0.9) + backdrop-blur(16px)
z-index: 40

标签样式:
  - 未选中: text-tertiary, 无背景
  - 选中: text-primary, 底部有 2px brand 色下划线
  - Hover: text-secondary

切换行为:
  - 点击标签 → 过滤画廊内容（前端过滤，无需请求）
  - 过渡动画: 图片 fade-in 200ms
```

### 4.5 无限滚动 / 加载

```
MVP 阶段只有 10 个模板，全部一次加载，无需分页。
V2 阶段模板增多后，改为 Intersection Observer 触底加载。
图片加载策略:
  - 首屏可见的图片: eager loading
  - 非首屏图片: loading="lazy"
  - 使用 next/image 组件，自动 WebP 转换和尺寸优化
```

```

  - background: rgba(40,40,44,0.98)
  - backdrop-filter: blur(24px)
  - border: 1px solid rgba(255,255,255,0.12)

## 8. [组件 7] 全局动态光效按钮

**所有 CTA 按钮（"开始创作"、"使用此模板"、"开始生成"、"发送"）都使用此效果。**

### 8.1 视觉描述

```
按钮有一圈微妙的光晕边框，光晕沿边框缓慢流动，营造高级感。

层次结构:
  1. 最底层: 按钮本体（brand 色背景 #C1272D，或渐变）
  2. 中间层: 边框光晕（conic-gradient 旋转动画）
  3. 最上层: 按钮文字（白色）
```

### 8.2 CSS 实现思路（给 Codex 的方向指引）

```
技术方案: 使用 CSS conic-gradient + animation 实现边框光流效果

外层容器:
  - position: relative
  - border-radius: 12px
  - padding: 1px（边框宽度）
  - background: conic-gradient(
      from var(--angle),
      transparent 0%,
      rgba(193,39,45,0.8) 10%,    // brand 色光
      rgba(255,255,255,0.3) 20%,   // 白色高光
      transparent 30%,
      transparent 100%
    )
  - animation: rotate-gradient 3s linear infinite
  - overflow: hidden

@keyframes rotate-gradient {
  from { --angle: 0deg; }
  to { --angle: 360deg; }
}

// 需要 CSS @property 注册 --angle 变量
@property --angle {
  syntax: '<angle>';
  initial-value: 0deg;
  inherits: false;
}

内层按钮:
  - background: #C1272D（实色，盖住 gradient 的中心）
  - border-radius: 11px（比外层小 1px）
  - 完全覆盖外层的 padding 区域以外的部分

效果: 看起来像一道红色+白色的光线沿着按钮边框缓慢旋转流动

Hover 态:
  - 光晕亮度增强（opacity 提升）
  - 按钮背景轻微变亮 (#D43B41)

Active/按下态:
  - scale(0.98)
  - 光晕加速旋转（1.5s 一圈）

Disabled 态:
  - 光晕停止旋转
  - 整体降低透明度 (opacity: 0.5)
  - cursor: not-allowed

Loading 态:
  - 按钮文字替换为 spinner + "处理中..."
  - 光晕变为脉冲效果（亮暗交替，而非旋转）
```

### 8.3 按钮尺寸变体

```

Medium (操作): height: 40px, padding: 0 20px, text: 14px, border-radius: 10px
Small (内联): height: 32px, padding: 0 16px, text: 13px, border-radius: 8px
Icon (发送): width: 36px, height: 36px, border-radius: 50% (圆形)
